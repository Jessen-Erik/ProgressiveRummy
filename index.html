How To Play Progressive Rummy

Goal: Finish 7 rounds with the lowest total score.

Players/Decks: 2-5 players use 2 decks, 6-7 players use 3 decks.

Deal: Rounds 1-4 deal 10 cards, rounds 5-7 deal 12 cards.

Turn Flow: 1) Current player may take discard. 2) If declined, eligible opponents may buy discard + penalty card. 3) Current player draws. 4) Play melds/layoffs. 5) Discard to end turn.

Sets: 3+ same rank (wild 2s allowed).

Runs: 4+ consecutive same suit (A can be low or high). Wild 2s can fill missing cards.

Wild Limits: Wilds cannot outnumber non-wilds in a meld. A set cannot be only wilds.

Round Requirements:

R1: 2 sets | R2: 1 set + 1 run | R3: 2 runs | R4: 3 sets

R5: 2 sets + 1 run | R6: 1 set + 2 runs | R7: 3 runs (and hand must be empty when submitted)

Scoring: Cards left in hand when a round ends are points against you: 2=20, A=15, K/Q/J=10, others face value.
